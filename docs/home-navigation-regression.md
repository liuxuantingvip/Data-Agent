# 首页导航与壳层回归用例

## 范围

- 页面：`/`
- 目标：首页品牌头、左侧导航、顶部栏、Hero 区、输入框下方按钮、与首页输入框边界相关的防回归链路

## 用例

- `NAV01` 品牌头显示
  - 左上品牌位显示新 Logo
  - 同时显示 `MData Agent`
  - 不再出现旧品牌名称

- `NAV02` 工作空间卡片移除
  - 左侧导航品牌头下方
  - 不存在 `我的工作空间`
  - 不存在工作区切换卡片

- `NAV03` 指令库文案
  - 左侧导航存在 `指令库`
  - 不再出现 `提示词库`

- `NAV04` 历史导航残留移除
  - 左侧导航不存在旧品牌导航项

- `NAV05` 历史 Banner 移除
  - 左侧导航下半区不存在任何旧品牌广告块
  - 不再出现历史推广文案

- `NAV06` 任务记录文案
  - 左侧历史区标题为 `任务记录`
  - 不再出现 `运行记录`

- `NAV07` 任务记录间距
  - `任务记录` 标题与记录列表之间无异常大留白
  - 历史区整体与导航区的垂直节奏自然

- `NAV08` 顶部栏裁剪
  - 顶部栏仅保留左侧侧栏折叠按钮和右侧用户头像
  - 不再出现历史营销按钮与旧品牌入口

- `NAV09` Hero 推广按钮移除
  - 标题上方不存在历史推广按钮

- `CMP01` 输入框下方按钮文案
  - 三个按钮均显示文字
  - 分别为 `@数据源`、`指令`、`附件`

- `CMP02` 数据源按钮功能
  - 点击 `@数据源`
  - 可正常打开数据源选择面板

- `CMP03` 首页能力胶囊不改输入框
  - 先输入普通文本
  - 点击首页能力胶囊
  - 输入框内容保持不变

- `CMP04` 输入框 `@` 选数据源不改卡片
  - 在输入框里输入 `@`
  - 选中一个数据源
  - 首页下方提示词卡片保持不变

- `CMP05` mention 面板层级
  - mention 面板展开后
  - 不被下方提示词卡片遮挡

- `CMP06` 示例卡片弹层
  - 点击第一张示例任务卡片
  - 弹层正常打开
  - 标题、`提示词(Prompt)`、`取消`、`查看回放`、`使用` 均可见

- `ENG01` 静态检查
  - `npm run lint` 通过

- `ENG02` 构建检查
  - `npm run build` 通过
