# 测试用例主文档

## 1. 目标与约束

- 本文档是当前唯一的测试基线文档。
- 目标覆盖三类验收：
  - 功能
  - 交互
  - 视觉
- 当前自动化约束固定为：
  - 单窗口
  - 保留 `linkfoxAgent` 基准页签
  - 单测试页签
  - 单 worker
  - 非并发

## 2. 优先级

- `P0`：主链路阻塞，失败不可发布
- `P1`：高频核心体验，失败需尽快修复
- `P2`：边界、空态、降级或一致性问题

## 3. 用例格式

每条用例固定包含：

- 用例 ID
- 优先级
- 页面/模块
- 场景名称
- 前置条件
- 步骤
- 预期结果
